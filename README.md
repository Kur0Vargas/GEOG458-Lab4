# \## Tile Set Descriptions

# \- \*\*Dark Theme:\*\* Helps visualize the Seattle area in a different contrast, highlighting certain areas distinctly from the basemap.  

# \- \*\*Satellite Theme:\*\* Provides a realistic view of Seattle using aerial imagery.  

# \- \*\*Street Theme:\*\* Allows users to familiarize themselves with streets and roads in a GPS app-like manner.  

# \- \*\*Outdoors Theme:\*\* Highlights streets and terrain with a different color hue for navigational use.
